# Ally360 Internal Docs

Sitio de documentación interna (onboarding + arquitectura + guías) construido con [Docusaurus](https://docusaurus.io/).

Requisitos: Node.js >= 20.

## Instalación

```bash
npm install
```

## Correr localmente

```bash
npm run start
```

## Build (opcional)

```bash
npm run build
```

Para servir el build localmente:

```bash
npm run serve
```
